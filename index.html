Respostas às perguntas do item 2:

a. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

Resposta: O atributo input especifica uma pequena dica que descreve o valor esperado de um campo de entrada (um valor de amostra ou uma breve descrição do formato esperado).

b. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

Resposta: O checked atributo é um atributo booleano.Quando presente, ele especifica que um <input> elemento deve ser pré-selecionada (marcada), quando a página é carregada. O checked atributo pode ser usado com <input type="checkbox"> e <input type="radio"> .

c. Na tag <input> para que serve o atributo "value"?

Resposta: O atributo especifica o valor de um elemento.

d. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Resposta: name <input type="radio">

e. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

Resposta: A tag define um rótulo para os seguintes elementos: 1. button 2. input 3. meter 4. output 5. progress> 6. select 7. textarea. O atributo for faz um tipo de link entre a label e o input. Logo o for tem que apontar para o ID do input.

f. Na tag <option> para que serve o atributo "selected"?

Resposta: Representa um controle que apresenta um menu de opções.

g. Nas tags de formulário para que serve o atributo "required"?

Resposta: Realiza a verificação na sumissão do formulário.

h. Para que serve a tag <textarea>?

Resposta: Controle de edição para uma caixa de texto.

i. Na tag <form> para que serve o atributo "action"?

Resposta: O atributo define a ação a ser executada quando o formulário é enviado.

j. Na tag <form> para que serve o atributo "method"?

Resposta: Define qual o método HTTP para enviar os dados (ele pode ser " GET " ou " POST ").

k. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

Resposta: O method="post". Nunca se deve usar o atributo GET se o seu formulário for conter dados confidenciais.

l. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

Resposta: O método HTTP padrão ao enviar dados de formulário é GET.

m. O que as tags <fieldset> e <legend> produzem?

Resposta: A tag <fieldset> é usada para agrupar elementos relacionados dentro de um formulário, desenhando uma borda em torno desses elementos. A tag <legend> representa um rótulo para o conteúdo de <fieldset>.
